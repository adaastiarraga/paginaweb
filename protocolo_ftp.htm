PROTOCOLO FTP

FTP (Protocolo de transferencia de archivos) es un protocolo de red estándar utilizado para la transferencia de archivos de un host a otro a través de una red basada en TCP, como Internet.

El FTP funciona abriendo dos conexiones que vinculan las computadoras que intentan comunicarse entre sí. Se designa una conexión para los comandos y respuestas que se envían entre los dos clientes, y el otro canal maneja la transferencia de datos. Durante una transmisión FTP, hay cuatro comandos utilizados por las computadoras, servidores o servidores proxy que se están comunicando. Estos son “enviar”, “obtener”, “cambiar directorio” y “transferir”.

Al transferir archivos, FTP utiliza tres modos diferentes: bloquear, transmitir y comprimir. El modo de flujo permite a FTP administrar información en una cadena de datos sin límites entre ellos. El modo de bloqueo separa los datos en bloques y, en el modo de compresión, FTP utiliza un algoritmo llamado Lempel-Ziv para comprimir los datos.
