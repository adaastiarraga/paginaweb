FUNCIONAMIENTO

Paso 1: reconocimiento de la URL introducida
El primer paso en el funcionamiento de la página web proviene del usuario, cuando introduce una dirección URL en la barra superior de su navegador web. O también cuando introduce una palabra clave en un buscador web y luego elige alguno de los enlaces ofrecidos, o simplemente hace click en un link ofrecido en alguna parte.   En todos los casos, el navegador se comunica con un servidor DNS, que opera como un listado de nombres y direcciones IP en internet, para dar con la ruta hacia el dominio solicitado, es decir, hacia donde está alojada la página web.
Paso 2: solicitud HTTP al servidor
En caso de que la dirección IP sea válida, el terminal del usuario se comunica con el servidor de destino, o sea, con la computadora o el conjunto de computadoras donde está alojada la información deseada. Luego, efectúa una solicitud HTTP o HTTPS, para que se establezca una comunicación basada en protocolos comunes y pueda iniciar la transmisión de la información contenida en el servidor.
Paso 3: respuesta del servidor
El servidor procesa la solicitud recibida y accede a los archivos que contienen instrucciones para el ensamblado de la página web, es decir, dónde va cada cosa y en qué otros servidores están, a su vez, las imágenes o elementos necesarios. Esto desencadena scripts y programas de asistencia que facilitan la localización rápida de los recursos y su transmisión hacia el terminal del usuario.
Paso 4: renderización de la página web
Con toda la información recibida, el terminal del usuario ensambla la página web, es decir, ubica en la pantalla del navegador cada cosa en su sitio. Entonces, está listo para recibir nuevas instrucciones del usuario, y el proceso se repite desde el comienzo.
